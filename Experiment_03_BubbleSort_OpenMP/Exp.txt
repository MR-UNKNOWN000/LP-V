# Parallel Bubble Sort using OpenMP – Complete Practical Explanation

---

# 1. Aim of the Program

To sort an array using:

1. Sequential Bubble Sort
2. Parallel Bubble Sort (Odd-Even Transposition Sort)

---

# 2. What is Bubble Sort?

Bubble Sort repeatedly compares two adjacent elements and swaps them if they are in the wrong order.

The largest element "bubbles up" to the end after each pass.

---

# 3. Example

Array:

```text
5 3 1 4 2
```

After sorting:

```text
1 2 3 4 5
```

---

# 4. Real-Life Example

Imagine arranging books by height:

* Compare two neighboring books.
* Swap if the left one is taller.
* Repeat until all books are in order.

# 6. Sequential Bubble Sort Function

```cpp
void bubbleSortSequential(vector<int>& a)
```

Sorts the array using the normal Bubble Sort algorithm.

---

## Outer Loop

```cpp
for (int i = 0; i < n - 1; i++)
```

Runs `n-1` passes.

---

## Inner Loop

```cpp
for (int j = 0; j < n - i - 1; j++)
```

Compares adjacent elements.

---

## Compare and Swap

```cpp
if (a[j] > a[j + 1])
```

If the left element is larger, swap them.

---

## Swap

```cpp
int temp = a[j];
a[j] = a[j + 1];
a[j + 1] = temp;
```

# 7. Parallel Bubble Sort Function

```cpp
void bubbleSortParallel(vector<int>& a)
```

Uses Odd-Even Transposition Sort, which is suitable for parallel processing.

---

# 8. What is Odd-Even Transposition Sort?

Instead of comparing all adjacent pairs in one pass:

### Even Phase

Compare:

```text
(0,1), (2,3), (4,5), ...
```

### Odd Phase

Compare:

```text
(1,2), (3,4), (5,6), ...
```

Repeat for `n` phases.

---

# 9. Outer Phase Loop

```cpp
for (int phase = 0; phase < n; phase++)
```

Runs `n` phases.

---

# 10. Even Phase

```cpp
if (phase % 2 == 0)
```

Process pairs:

```text
(0,1), (2,3), (4,5)
```

---

# 11. Odd Phase

```cpp
else
```

Process pairs:

```text
(1,2), (3,4), (5,6)
```

---

# 12. Why This Is Parallel-Friendly

Pairs do not overlap.

Example:

* (0,1)
* (2,3)
* (4,5)

Each pair can be processed independently.

> In a full OpenMP implementation, these loops would use:

```cpp
#pragma omp parallel for
```

# 21. Time Complexity

Sequential Bubble Sort:

O(n^2)

Parallel Odd-Even Sort:

O(n^2)

Although still O(n²), parallel execution reduces actual running time.

---

# 22. Space Complexity

O(1)

Only one temporary variable is used for swapping.

(Overall program uses O(n) because arrays are stored in vectors.)

---

# 23. Sequential vs Parallel Bubble Sort

| Sequential                  | Parallel                                |
| --------------------------- | --------------------------------------- |
| Compares one pair at a time | Compares multiple non-overlapping pairs |
| Normal Bubble Sort          | Odd-Even Transposition Sort             |
| Single thread               | Multiple threads possible               |

---

# 24. Why Bubble Sort Is Called “Bubble”

The largest element moves to the end like a bubble rising to the top.

Yes.

### Q5. What is Odd-Even Transposition Sort?

A parallel version of Bubble Sort.

To compare independent pairs.
